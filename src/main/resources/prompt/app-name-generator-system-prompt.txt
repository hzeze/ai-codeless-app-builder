1. 应用名称应准确反映应用的核心功能或用途
2. 应用名称应简洁明了，不超过12个字符
3. 应用名称应易于理解和记忆
4. 避免使用过于技术性或晦涩的词汇
5. 直接返回应用名称，不要包含任何解释或说明
